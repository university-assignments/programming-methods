# Вопросы к защите отчета по работе (ответы приводятся в отчете).

1)	В чем заключается смысл операций  = и  = =  в программе на языке С++ и отличие их по результатам?
2)	Как записываются на языке С++ математические функции:
	a.	Натуральный логарифм, десятичный логарифм, двоичный логарифм.
	b.	Корень квадратный, корень N-ой степени, N-я степень
	c.	Экспонента
	d.	Модуль вещественного и целого числа
3)	Какие математические константы содержатся в математической библиотеке языка и как они обозначаются в программе? Для чего они используются?
4)	Какие базовые типы данных вам известны, для чего нужно их разнообразие?
5)	Для чего требуется тестирование программы? Какие средства отладки программ вам известны? В чем разница между отладкой и тестированием?

# Ответы

1) Операция `=` используется для присваивания значения переменной, а `==` — для сравнения двух значений на равенство.
2) a) `log(x)` — натуральный логарифм, `log10(x)` — десятичный, `log2(x)` — двоичный; b) `sqrt(x)` — квадратный корень, `pow(x, 1.0/n)` — корень n-ой степени, `pow(x, n)` — n-я степень; c) `exp(x)` — экспонента; d) `fabs(x)` — модуль вещественного, `abs(x)` — модуль целого.
3) В математической библиотеке C++ содержатся константы, такие как `M_PI` (число π) и `M_E` (число e), которые используются для математических вычислений.
4) Базовые типы данных: `int`, `float`, `double`, `char`, `bool`; их разнообразие нужно для эффективного использования памяти и точности вычислений.
5) Тестирование требуется для проверки корректности программы; средства отладки: отладчики (например, GDB), логирование; отладка — поиск и исправление ошибок, тестирование — проверка работы программы на соответствие требованиям.
